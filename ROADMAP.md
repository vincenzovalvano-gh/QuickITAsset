# QuickITAsset Roadmap

This document outlines the future development plans for QuickITAsset.

## Upcoming Features

### 1. Centralized Multi-Site Management
- **Goal**: Enable management of multiple remote sites from a single dashboard.
- **Features**:
    - Agent-based architecture for remote scanning.
    - Centralized server/database to aggregate results from multiple agents.
    - Web-based dashboard for viewing and managing all sites.
    - Secure communication (TLS/SSL) between agents and server.

### 2. Vulnerability Scanning
- **Goal**: Integrate basic vulnerability assessment capabilities.
- **Features**:
    - Integration with CVE databases (e.g., NVD).
    - Identification of outdated services and operating systems.
    - Basic port scanning for common vulnerabilities (e.g., open RDP, SMB).
    - Reporting on potential security risks per asset.

## Future Ideas
- Network Topology Mapping (Visual Map).
- Alerting System (Email/Webhook notifications for new devices or down hosts).
- Integration with CMDBs.
